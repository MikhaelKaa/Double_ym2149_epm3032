Доработки для платы версии 1.5 за авторством ponchick (ttps://github.com/ponchick/TS_Ext_EPM3032)
Доработки связанные с тактированием Ямах.
1. Разрезать дорожку от ламели по которой идет клок из шины (на схеме обозначена как b8)
2. Ламель b8 соеденить с U1-43
3. Вывод U1-18 соеденить с U3-22
4. В любом удобном месте разместить переключатель, один полюс которого соеденен с землей, другой через резистор 1к с 3.3В, 
 средную точку подключить к U1-1. Переключая можно выбрать делитель частоты тактирования ямах (без деления, деление на два - для работы в режиме турбо)
Доработки связанные с дешифрацией (совместимость с портом dffd и просто корректная работа).
1. Ламель b11 (сигнал А2) соеденить с U1-44
2. Ламель b12 (сигнал А3) соеденить с U1-26
3. Ламель а18 (сигнал RD) соеденить с U1-37
4. Ламель а2 (сигнал А13) соеденить с U1-28
